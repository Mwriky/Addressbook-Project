# 📒 Address Book Management System (C Project)

## 📌 Overview
This is a console-based Address Book Management System developed using C programming language.  
The project demonstrates structured programming, file handling, pointers, and modular design concepts.

---

## 🚀 Features

- ➕ Add New Contact
- 🔍 Search Contact
- ✏ Edit Existing Contact
- ❌ Delete Contact
- 📄 Store contacts using File Handling
- 🧠 Modular code structure using multiple source and header files

---

## 🛠 Technologies Used

- C Programming Language
- GCC Compiler
- File Handling
- Structures
- Pointers
- Command Line Arguments

---

## ▶ How to Compile & Run
### 🔹 Compile
gcc *.c -o addressbook

### 🔹 Run
./addressbook
